HP

High Power Interface Tolerance Evaluation System

Embedded C Flutter AWS Flask

Senior Design Capstone at Texas A&M University in College Station, completed from August 2019 to May 2020. I served as the sole software engineer on a 4-person team building a safety-critical intermediary system for evaluating tolerance thresholds between high-power electrical devices.

Role

  • Owned the full software architecture across embedded firmware, mobile application, and cloud backend
  • Designed software around a hardware risk profile that required fault tolerance, redundancy, and predictable control behavior
  • Coordinated software interfaces with the rest of the team so the mobile app, embedded module, and test workflow could operate as one system

System

  • RTOS-based embedded system in C for real-time hardware control
  • Bluetooth connectivity for mobile communication, plus I²C and additional hardware communication protocols
  • Flutter mobile app for iOS and Android to serialize tests, format customer data, and communicate with the hardware module
  • Containerized AWS backend using Flask and PostgreSQL as the central data layer between mobile and hardware workflows

Tech Stack

Embedded C RTOS Bluetooth I²C Flutter iOS Android Flask PostgreSQL Docker AWS
